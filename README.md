# Neon Grid

A theme inspired by a certain 80's film with "The Grid" — deep blue-black lit by a full neon spectrum
(fuchsia · green · orange · cyan · blue · violet · red · yellow). Glowing controls, gradient
hairlines, letter-spaced display type over terminal mono. Grid kept; scanlines dropped.

## Available now

- **[Android — Material 3](android/)** — a complete, drop-in theme that styles virtually every
  Material + platform widget and **recolors the whole system from four theme attributes**
  (`colorPrimary` / `colorSecondary` / `colorTertiary` / `ngAccent`). See
  **[android/README.md](android/README.md)**.

More platforms (Compose, Web/CSS, SwiftUI, Flutter) are planned against the same shared tokens.
